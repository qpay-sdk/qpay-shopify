# Privacy Policy - QPay Payment Gateway for Shopify

## Data Collection

The QPay Payment Gateway app collects the following information during payment processing:

- **Order ID**: Used to create and track payment invoices
- **Order Amount**: Used to generate the correct payment amount
- **Customer Email**: Optionally used as invoice receiver code

## Data Usage

All collected data is used solely for the purpose of processing payments through the QPay V2 payment system. No customer data is stored permanently by this application.

## Data Sharing

Payment data is transmitted to QPay (https://merchant.qpay.mn) for invoice creation and payment verification. No data is shared with any other third parties.

## Data Retention

This application does not retain any customer personal data beyond the active payment session. All payment data is stored by the Shopify platform and by QPay according to their respective data retention policies.

## GDPR Compliance

This application implements the required Shopify GDPR endpoints:

- **Customer Data Request**: Returns confirmation that no customer data is stored
- **Customer Data Erasure**: Confirms no customer data needs to be erased
- **Shop Data Erasure**: Confirms cleanup on app uninstall

## Contact

For privacy-related inquiries, contact: https://github.com/qpay-sdk/qpay-shopify/issues
